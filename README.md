# Tilt Warning System for Portable Devices

## Project Description
This project uses an STM32 microcontroller and MPU6050 sensor to detect unsafe tilt angles in portable devices or cargo boxes.

When the tilt exceeds a critical angle:
- LED warning turns ON
- Warning message is sent through UART

## Components Used
- STM32 Nucleo-L031K6
- MPU6050
- LED
- 220Ω resistor

## Features
- Real-time tilt detection
- UART warning messages
- LED alert system
- I2C communication with MPU6050
